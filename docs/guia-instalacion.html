📱 Servidor PHP en Android con Termux

Convierte tu celular o tablet Android en un servidor PHP completo para ejecutar Query Manager sin necesidad de una PC.

┌─────────────────────────────────────┐ │ Tablet/Celular Android │ │ │ │ ┌─────────┐ ┌────────────────┐ │ ┌──────────────┐ │ │ Chrome │───▶│ Termux │ │ │ MySQL/SQL Srv│ │ │ (naveg.)│ │ Apache + PHP │──┤─WiFi─▶│ (otro equipo)│ │ └─────────┘ │ Puerto 8080 │ │ └──────────────┘ │ └────────────────┘ │ │ http://localhost:8080/phpadmin/ │ └─────────────────────────────────────┘

Otros dispositivos en la misma red tambien pueden acceder usando la IP del tablet.

✅ Requisitos

  • Android 7.0 o superior
  • Minimo 2GB de RAM disponible
  • ~500MB de espacio libre
  • Conexion a la misma red donde estan las bases de datos
1
Instalar Termux
▼
⚠️
NO instalar desde Google Play — la version esta desactualizada y tiene errores.
  1. Descargar desde F-Droid (tienda de apps libre):
    Abrir en el navegador del celular: https://f-droid.org/en/packages/com.termux/
  2. Descargar e instalar el APK.
  3. Si pide permiso para "instalar desde origenes desconocidos", aceptar.
  4. Abrir Termux. Aparece una terminal negra.
2
Actualizar paquetes e instalar PHP + Apache
▼

Copiar y pegar estos comandos uno por uno en Termux:

# Actualizar repositorios
pkg update -y && pkg upgrade -y

# Instalar PHP, Apache, git y herramientas
pkg install -y php-apache apache2 git zip unzip

# Instalar extensiones PHP necesarias
pkg install -y php-pdo php-openssl
3
Configurar Apache
▼

Abrir el archivo de configuracion:

nano $PREFIX/etc/apache2/httpd.conf

Buscar y modificar estas lineas (usar Ctrl+W para buscar en nano):

# Cambiar el puerto (Android no permite el 80 sin root)
Listen 8080

# Habilitar PHP - buscar la linea de LoadModule php y descomentarla
LoadModule php_module libexec/apache2/libphp.so

# Buscar DirectoryIndex y agregar index.php
DirectoryIndex index.php index.html

Guardar con Ctrl+O, Enter, Ctrl+X.

4
Descargar Query Manager
▼
# Ir a la carpeta web de Apache
cd $PREFIX/share/apache2/default-site/htdocs

# Opcion A: Si tienes el codigo en git
git clone <tu-repositorio> phpadmin

# Opcion B: Copiar desde el almacenamiento del celular
# Primero dar permiso de almacenamiento a Termux:
termux-setup-storage
# Luego copiar:
cp -r ~/storage/downloads/phpadmin ./phpadmin

# Dar permisos de escritura a las carpetas de datos
chmod -R 777 phpadmin/data
chmod -R 777 phpadmin/config
5
Ajustar la app para Termux
▼

Editar el .htaccess porque Apache en Termux tiene una estructura diferente:

nano phpadmin/.htaccess

Verificar que el contenido sea:

RewriteEngine On
RewriteBase /phpadmin/

# Proteger directorios sensibles
RewriteRule ^data/ - [F,L]
RewriteRule ^config/ - [F,L]

# API routing
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^api/(.*)$ api/index.php [QSA,L]
6
Iniciar el servidor
▼
# Iniciar Apache
apachectl start

Abrir Chrome en el mismo celular y navegar a:

http://localhost:8080/phpadmin/
💡
Para acceder desde otro dispositivo en la misma red: 
ifconfig wlan0 | grep "inet "
Luego usa: http://192.168.X.X:8080/phpadmin/
7
Script de inicio rapido
▼

Crear un script para arrancar el servidor con un solo comando:

# Crear el script
cat > ~/start-qm.sh << 'SCRIPT'
#!/data/data/com.termux/files/usr/bin/bash
echo "================================"
echo "  Query Manager - Servidor PHP"
echo "================================"

# Iniciar Apache
apachectl start 2>/dev/null

# Obtener IP
IP=$(ifconfig wlan0 2>/dev/null | grep "inet " | awk '{print $2}')

echo ""
echo "Servidor iniciado correctamente"
echo ""
echo "Acceso local:  http://localhost:8080/phpadmin/"
if [ -n "$IP" ]; then
    echo "Acceso en red:  http://$IP:8080/phpadmin/"
fi
echo ""
echo "Para detener: apachectl stop"
echo "================================"
SCRIPT

# Dar permisos de ejecucion
chmod +x ~/start-qm.sh
⚡
Ahora para iniciar el servidor solo ejecuta: ~/start-qm.sh
8
Inicio automatico al abrir Termux Opcional
▼

Para que el servidor arranque automaticamente cada vez que abres Termux:

echo "~/start-qm.sh" >> ~/.bashrc
9
Crear acceso directo en Android Opcional
▼
  1. Instalar Termux:Widget desde F-Droid.
  2. Crear la carpeta de shortcuts: 
    mkdir -p ~/.shortcuts
cp ~/start-qm.sh ~/.shortcuts/QueryManager
  3. En la pantalla de inicio del celular, agregar un widget de Termux.
  4. Aparecera "QueryManager" como opcion. Al tocarlo, inicia el servidor automaticamente.

Crear la red en ZeroTier
▼
  1. Ir a https://my.zerotier.com/ y crear una cuenta gratuita.
  2. Hacer clic en "Create A Network".
  3. Se crea una red con un Network ID de 16 caracteres (ej: a09acf023389e132). Anotar este ID.
  4. En la configuracion de la red:
    • Name: Poner un nombre descriptivo (ej: "QueryManager VPN").
    • Access Control: Dejar en "Private" (los dispositivos deben ser aprobados manualmente).
    • IPv4 Auto-Assign: Asegurarse de que este habilitado (rango por defecto: 10.147.20.0/24).

📤 Instalar Query Manager como App (PWA)

Query Manager ya incluye soporte PWA. Esto permite instalarlo como una app nativa en cualquier dispositivo.

📱 En Android (Chrome)

  1. Abrir la URL de Query Manager en Chrome.
  2. Aparece un banner automatico: "Agregar Query Manager a pantalla de inicio".
  3. Tocar Instalar.
  4. Se crea un icono en el escritorio, igual que una app nativa.
  5. Al abrirla: pantalla completa, sin barra de URL.

🔒 Importante: PWA requiere contexto seguro

Chrome solo permite instalar PWA en localhost o HTTPS. Si accedes por IP (ej: http://192.168.1.50), el boton de instalar no aparecera.

Solucion rapida: En Chrome del celular, abre chrome://flags, busca "Unsafely treat insecure origin as secure", agrega la URL de tu servidor y activa. Reinicia Chrome.

🍏 En iPhone/iPad (Safari)

  1. Abrir la URL en Safari.
  2. Tocar el boton Compartir (cuadrado con flecha).
  3. Tocar "Agregar a pantalla de inicio".
  4. Se instala con icono y nombre.

💻 En PC/Laptop (Chrome/Edge)

  1. Abrir la URL de Query Manager.
  2. En la barra de direccion aparece un icono de instalar.
  3. Se instala como ventana independiente (sin pestanas del navegador).

⚡ Flujo completo con ZeroTier + PWA

1. Encender ZeroTier en el celular (siempre activo en segundo plano) 2. Tocar el icono de Query Manager en la pantalla de inicio 3. La app se abre en pantalla completa 4. Iniciar sesion y trabajar normalmente No necesitas: ❌ Configurar IP cada vez (ZeroTier mantiene la misma IP siempre) ❌ Estar en la misma WiFi (funciona desde cualquier red) ❌ Puerto abierto en el router (ZeroTier perfora NAT) ❌ Pagar nada (gratis hasta 25 dispositivos)

📤 Problemas con la PWA

ProblemaSolucion
