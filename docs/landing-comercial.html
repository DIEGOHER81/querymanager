Caracteristicas
Todo lo que necesitas para gestionar tus datos
Una solucion completa que reemplaza multiples herramientas. Sin instalaciones, sin dependencias, desde cualquier dispositivo.

Multi-Conexion

Conecta simultaneamente a multiples servidores MySQL y SQL Server. Guarda y organiza todas tus conexiones con credenciales cifradas.

Explorador de Esquemas

Navega bases de datos, tablas, vistas, procedimientos y funciones. Visualiza columnas, parametros y definiciones de rutinas.

Editor SQL Avanzado

Ejecuta consultas directas o via stored procedures con formato JSON. Cancela queries largas, ve resultados en tiempo real.

Exportacion Profesional

Exporta resultados a CSV, Excel (XLSX nativo) o JSON con metadatos. Todo generado en el servidor sin librerias externas.

Auditoria Completa

Registro automatico de cada consulta ejecutada: conexion, SQL, tiempo, filas, estado. Filtra, busca y marca favoritas.

Seguridad Empresarial

AES-256-CBC para credenciales, bcrypt para passwords, CSRF, headers de seguridad, roles admin/user, cambio obligatorio de contrasena.

PWA Instalable

Instala como app nativa en cualquier dispositivo. Funciona offline con interfaz cacheada. Responsive en movil y escritorio.

Cero Dependencias

PHP puro + JavaScript vanilla. Sin Composer, sin npm, sin frameworks. Sube los archivos al hosting y funciona.

Multi-Query Simultaneo

Ejecuta la misma consulta en multiples servidores a la vez. Resultados por tabs, exportacion independiente. Ideal para comparar datos entre ambientes.

Cross-Connection JOIN

Cruza datos de servidores completamente diferentes como si fueran uno solo. Constructor visual + Editor SQL libre con motor de consultas virtual en memoria.

Comparador de Esquemas

Compara tablas, columnas, vistas y procedimientos entre dos bases de datos. Genera scripts de migracion automaticos (CREATE, ALTER, DROP) con ejecucion controlada.

Intellisense y Autocompletado

Sugerencias inteligentes de tablas, columnas y keywords mientras escribes. Explorador de objetos de BD integrado en todos los editores SQL.
